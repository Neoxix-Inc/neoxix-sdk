# Code of Conduct

## Our pledge

We pledge to make participation in this project respectful, inclusive, and free from harassment.

## Expected behavior

Use welcoming language, accept constructive feedback, focus on engineering outcomes, and respect differing experiences and viewpoints.

## Unacceptable behavior

Harassment, discrimination, threats, sexualized attention, doxxing, and deliberate disruption are not tolerated.

## Enforcement

Report conduct concerns privately to repository maintainers through an appropriate private GitHub channel. Maintainers will review reports fairly, protect privacy where possible, and may remove content or restrict participation when necessary.

This policy applies in project spaces and when representing the project publicly.
